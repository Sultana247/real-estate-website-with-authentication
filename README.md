Real estate website using react, tailwind css, swiper, firebase authentication, firebase hosting.
Features
1. login with email/password, sign in with google, sign in with GitHub(based on firebase)
2. Registration form and register user successfully
3. After login user profile picture will be shown on header section beside the logout button. and when hover on the profile picture user name will be popup
4. banner using swiper 3 slides available
5. Private route is estates the details page. when click any of the estate then if the user logged in the details page will be redirected otherwise redirected to the login page
